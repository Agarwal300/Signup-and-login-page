# 🔐 Sign Up & Login Page (HTML & CSS)

A simple and responsive **Sign Up and Login page** built using only **HTML and CSS**.  
This project focuses on UI design and layout without using JavaScript or backend.

---

## 🚀 Features

- 📝 Clean Sign Up form
- 🔑 Login form UI
- 🎨 Modern and responsive design
- 📱 Mobile-friendly layout
- ⚡ Pure HTML & CSS (no JavaScript)

---

## 🛠️ Tech Stack

- HTML5
- CSS3

---

## 📂 Project Structure

├── index.html

├── style.css

└── README.md


---

## ⚙️ How to Run

1. Download or clone the repository: https://github.com/Agarwal300/signup-login-page.git 
2. Open the project folder
3. Double click on `index.html`  
   OR open it in your browser

---

## 💡 How It Works

- This project contains static forms for:
  - User Sign Up
  - User Login
- It is designed only for **frontend practice**
- No data is stored (no backend)
